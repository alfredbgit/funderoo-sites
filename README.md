# funderoo-sites

Static site source files for Funderoo Labs projects.

## Structure

```
docs/              → IntelPulse product page (GitHub Pages serves this at root)
funderoo-labs/     → Funderoo Labs company page
```

## GitHub Pages

IntelPulse product page is served from `docs/` via GitHub Pages.

**Repo Settings → Pages → Source: `main` branch, `/docs` folder.**

GitHub Pages URL: `https://alfredbgit.github.io/funderoo-sites/`

## Updating

Push changes to `main`. GitHub Pages auto-deploys from `docs/`.
