# funderoo-sites

Static site source files for Funderoo Labs projects.

## Structure

```
intel-pulse/product-page/  → IntelPulse product page (auto repair vertical)
funderoo-labs/             → Funderoo Labs company page
```

## GitHub Pages

Each subdirectory can be deployed as a GitHub Pages site:
- Set publishing source to the subdirectory in repo Settings → Pages
- Or: push subdirectory to a dedicated branch (e.g. `gh-pages`)

## Deploy IntelPulse Product Page

In repo Settings → Pages → Source:
- Branch: `main`, folder: `/intel-pulse/product-page`
- Or push this repo to `gh-pages` branch and serve from root
